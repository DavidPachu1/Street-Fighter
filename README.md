# 🕹️ Street Fighter - Player Registration

¡Bienvenido al torneo! Este proyecto es una página web temática inspirada en el universo de **Street Fighter**. Incluye una interfaz de selección de personajes clásicos y un formulario completo para la inscripción de luchadores.

## 🚀 Características principales
*   **Character Select:** Una rejilla visual con 16 luchadores icónicos (Ryu, Ken, Chun-Li, Guile, etc.) con sus nombres y retratos clásicos.
*   **Formulario de Inscripción:** Sistema de registro temático que incluye:
    *   Datos personales y de contacto.
    *   Selección de género y fecha de nacimiento.
    *   Selector específico de personaje favorito.
    *   Área de comentarios para la estrategia de combate.
*   **Diseño Inmersivo:** Fondo de pantalla completa con el escenario clásico del templo japonés y tipografía estilo arcade.

## 🛠️ Tecnologías
*   **HTML5:** Para la estructura de la tabla de personajes y la creación del formulario complejo.
*   **CSS:** Estilizado de contenedores, transparencia en el formulario y posicionamiento de elementos sobre el fondo dinámico.
